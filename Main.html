Nux=@(b1,b2,b3,AT) [b1 0 b2 0 b3 0]/2/AT; % shape function derivative

Nvx=@(b1,b2,b3,AT) [0 b1 0 b2 0 b3]/2/AT;

Nuy=@(a1,a2,a3,AT) [a1 0 a2 0 a3 0]/2/AT;

A=zeros(6,6);

AA =zeros(6,6,nt);

B = AA;

for i=1:nt

% A = (Nux.')*Nvy -(Nuy.')*Nvx;

% This is related to J=det(F) where F is the deformation tensor

Mesh-independency filter (Density filtering)

A fundamental aspect of the TO is the distinction between the physical densities $\rho$ and the merely numerical ones $x$.

Here the method suggested by Bruns and Tortorelli has been implemented:

where is the weight associated with the $i^{th}$ element within the prescribed neighborhood of the $j^{th}$ one, determined as:

Here, , called filter radius, is the radius of the specified neighborhood and the distance between the centroids of the two elements. This filter makes the results independent from the adopted mesh, improving the resolution as decreases. However, the maximal resolution is linked to the mesh itself, since the minimum applicable cannot be smaller than elements' size.

%% Mesh-independency filter preparation

rmin = 6; %depends on the mesh-size (it's not a contradiction with the section title)

xG = (x1+x2+x3)/3;

yG = (y1+y2+y3)/3;

i=1:nt;

[I,J]=meshgrid(i,i);

W = max(0,rmin-sqrt((xG(J)-xG(I)).^2+(yG(J)-yG(I)).^2));

W = W./repmat(sum(W,2),1,nt);

W = sparse(W);

tail_part = find(xG>0.85*corde);

Final deformed structure

A final FE-Analysis is performed, and the coordinates of the points in the final configuration are stored in the matrix pp.

rho = W*xval; % density filtering

pp=p;

end
